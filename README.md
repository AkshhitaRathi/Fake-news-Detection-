# Fake News Detection System

This project detects whether news is real or fake using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP
- Flask

## Features
- Detects fake news
- User input prediction
- Machine learning model trained on dataset

## Project Files
- app.py
- model.pkl
- requirements.txt

## How to Run
1. Install requirements
2. Run app.py
3. Open localhost in browser
